## Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/jinalegwu/softwaresfast.git
```

2. Navigate to the project directory:
```bash
cd softwaresfast
```

3. Open the `index.html` file in your browser:
```bash
open index.html
```

Or use a local server:
```bash
python -m http.server 8000
# Then visit http://localhost:8000
```

## Deployment

### Option 1: GitHub Pages
1. Push code to `main` branch
2. Go to repository Settings → Pages
3. Select `main` branch as source
4. Site will be available at `https://jinalegwu.github.io/softwaresfast`

### Option 2: Traditional Hosting
1. Upload all files to your hosting provider
2. Ensure `index.html` is in the root directory
3. Access through your custom domain

### Option 3: Deploy with Custom Domain
1. Purchase domain (e.g., softwaresfast.com)
2. Point domain DNS to hosting provider
3. Upload website files
4. Configure domain settings in hosting control panel

## Customization Guide

### Change Brand Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f5576c;
    /* ... more colors ... */
}
```

### Update Content
1. Edit company name, description, and services in `index.html`
2. Update contact information in the contact section
3. Replace portfolio projects with actual company work
4. Update social media links

### Add More Sections
1. Add new HTML section in `index.html`
2. Create corresponding CSS in `styles.css`
3. Add navigation link in the navbar
4. Add smooth scroll behavior in `script.js` (already automatic)

### Modify Portfolio
Replace portfolio items with your actual projects:
- Update project titles and descriptions
- Change technology tags
- Modify gradient backgrounds
- Update project icons
